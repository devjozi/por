# POR (Personal Opportunity Router)

Standalone static site repository for the POR landing page.

## Deploy

This repo deploys to GitHub Pages using GitHub Actions.

1. Push to `main`
2. In GitHub: `Settings -> Pages -> Build and deployment -> Source = GitHub Actions`
3. The workflow publishes the site automatically

## Versioning

- `VERSION` is the canonical version marker
- Optional tags should use the format `vX.Y.Z`
